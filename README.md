## Features
- **Lv.50 PvP-Focused**: Defaults to Lv.50 for quick PvP calculations (PokeMMO's PvP standard)

- **Enhanced EV System**: Visualizes EV distribution and remaining Evs: `Used EVs/510(Remaining EVs)`
